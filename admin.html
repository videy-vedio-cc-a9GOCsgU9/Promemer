How to publish

This site uses a simple data/items.json file to store all your content. No database needed.

  1. Fill in the form and click Generate JSON entry.
  2. Copy the generated entry.
  3. Open data/items.json in your GitHub repo.
  4. Add the new entry inside the [ ] array (add a comma between entries).
  5. Commit the file. Cloudflare Pages auto-deploys in ~30 seconds.
💡 Upload your actual files (images, videos, zips) directly to the GitHub repo under a files/ folder. Then use the raw GitHub URL as the File URL above.
